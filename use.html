기본적으로 system이라는 함수가 있습니다. 함수 속성들에 종류는 이렇습니다.

이름 사용법 작동 되는것
input system.go.input(텍스트) | 텍스트 부분에 input할때 물어볼것을 넣는다. input내용과 받을수 있다.
output system.go.opuput(텍스트) | 텍스트 부분에 출력할걸 넣는다. 출력한다.
reset system.go.reset() 전체 코드를 처음부터 시작한다.

그다음은 변수 인데 변수는 간단하게 파이썬처럼 저장합니다. 하지만 js처럼 저장도 할수있습니다.

배열은 []에다가 저장합니다. 그리고 뒤에 <>가 붙는데 <0>하면 인덱스가 0부터 시작하고 <1>하면 1부터 시작합니다.

딕셔너리는 {}에다가 키 : 값 형식으로 저장합니다. 똑같이 <>로 인덱스 시작값을 설정합니다.

if,while은 js와 똑같지만 for 문은 for i<1> 10 {} 이런식으로 하면 되는데 i부분은 반복횟수고 <1>은 배열이나 딕셔너리처럼 한거와 같고 그다음에 반복횟수가 붙습니다.

모듈은 불러올때는 #out_module <모듈이름> 이런식입니다.

함수를 만들때는 class basic function 이름(매개변수) {} 이런식이다.

Math함수가 있다. 속성을 보여주겠다.

이름 사용법 작동 되는것
random math.random(첫, 둘) | 첫번째 인자부터 두번째 인자까지에 랜덤숫자를 재시해준다. 랜덤숫자를 얻는다.
Max math.max(...) | 인자에 수를 넣으면 된다. 인자에 넣은 수중 가장 큰 수를 반환한다.
min math.min(...) | 인자에 수를 적으면된다. 인자에 넣은 수중 가장 작은수를 반환한다.

string 함수도 있다. 속성을 보여주겠다.

이름 사용법 작동 되는것
split string.split(문자, 분활기준) | 첫번째에 문자 두번째에 분환기준. 분환기준에 따라 문자를 분리 해준다. 분활기준은 사라지고 배열로 반환된다.
find string.find(문자, 찾을것) | 첫번째에 문자 두번쨰에 찾을거. 문자에서 찾을것을 찾아주고 배열형식으로 첫번째에는 문자를 찾았는데 true 나 false로 해주고 두번째에는 몇번째몇번째에서 찾았는지 세번째에서는 몇개 찾았는지를 반환해준다.
re string.re(문자, 불린) | 첫번째에 문자 두번쨰에 불린을 넣는다. 불린이 false면 문자에 문자를 넣으면 숫자로 바꿔주고 숫자를 넣으면 문자로 바꿔준다. true라면 문자를 거꾸로 바꿔는데 만약 sdd라면 dds가 되는거다.
